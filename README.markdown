# grade_runner

A Ruby client for [firstdraft Grades](https://grades.firstdraft.com)


## Installation

Add this line to your application's Gemfile:

```ruby
gem "grade_runner", github: "firstdraft/grade_runner"
```

And then execute:
```bash
$ bundle
```
